This is a Qr Code generator web app. It is built using NextJS and TailwindCSS. It is deployed on Vercel.

not yet

## Run Locally

Clone the project

```bash
  git clone
```

Go to the project directory

```bash
  cd qrcode-generator
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```

## Tech Stack

**Client:** NextJS, TailwindCSS

**Server:** Vercel
